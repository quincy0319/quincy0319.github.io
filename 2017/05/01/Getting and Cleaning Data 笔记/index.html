Getting and Cleaning Data

Week 1

1. 整洁数据（Tidy Data）的组成

  1. 原始数据
  2. 整洁数据集
  3. 描述每个变量及其在整洁数据集中值的密码本（Code Book），也称作Meta Data
  4. 从第一步到第二步到第三步的详细过程

2. 原始数据的原则

  1. 没有在数据上运行任何软件
  2. 没有篡改任何数据
  3. 没有移除任何数据
  4. 没有总结任何数据

3. 整洁数据的原则

  1. 同一个变量放在一列中
  2. 每个不同的观测分别在一行中
  3. 每个变量都应该有一个表格
  4. 如果有多个表格，必须保证每个表格都有能使它们链接起来的列

4. 数据处理脚本的原则

输入同样的原始数据，应当输出同样的整洁数据

5. 工作目录

  1. 常用指令

  2. 检查与创建目录

9. data.table包

由C编写，比data.frame速度更快

  1. tables()
    所有数据的名称、行数、列数、所占空间大小、键
  2. 取子集
    * DT[2, ]
* DT[DT$y == "a", ]
* DT[c(2, 3), ] # 默认按行取子集
  3. 取子集时可以将想要执行的函数写成列表（list）放进去
    * DT[, list(mean(x), sum(z))]
* DT[, table(y)]
  4. 添加新列
    * DT[, w : = z ^ 2]
* DT[, m : = {tmp <- (x+z); log2(tmp + 5)}]
* DT[, a: = x > 0]
  5. 通过键（keys）来加快处理速度
